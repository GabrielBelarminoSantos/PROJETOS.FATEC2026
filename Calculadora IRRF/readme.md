# 💸 Calculadora IRRF

projeto desenvolvido com **HTML, css e javascript** com o objetivo de elaborar uma calculadora que forneça os valores a serem descontados (INSS e IRRF) do salário bruto do contribuinte e o valor do salário liquido.
O projeto inclui CSS e manipulação de campo via JS e faz parte dos projetos criados na FATEC-Itu com o intuito de aprimoramento de habilidades de linguagem de programação.

> Curso Analise e Desenvolvimento de Sistemas - Fatec Itu

## 📝 Tecnologias Utilizadas
-HTML
-CSS3
-JAVASCRIPT (Vanilla JS)
